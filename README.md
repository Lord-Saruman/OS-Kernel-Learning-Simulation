# OS Kernel Learning Simulation

An interactive simulation project to understand and learn core Operating System kernel concepts.

## Overview

This project is designed as a hands-on learning tool for exploring OS kernel internals, including:

- Process Management & Scheduling
- Memory Management
- File System Concepts
- Inter-Process Communication (IPC)
- Synchronization & Deadlocks

**Subject:** Operating Systems  
**Semester:** 4 — Spring 2026
